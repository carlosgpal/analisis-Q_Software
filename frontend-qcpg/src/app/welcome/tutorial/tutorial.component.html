Bienvenido a QCPG

QCPG es una herramienta de análisis de software cuántico estático, diseñada para optimizar y asegurar tus programas cuánticos. Utilizando el estándar Code Property Graph (CPG), QCPG integra múltiples aspectos del análisis de código en una sola representación unificada, permitiendo un análisis más profundo y eficiente.

Subdivisiones de CPG

  • Abstract syntax tree (AST): Facilita la representación de la estructura gramatical de tu código.
  • Control-flow graph (CFG): Visualiza todas las rutas posibles que puede tomar la ejecución del programa.
  • Program dependence graph (PDG): Muestra las dependencias entre los nodos del CFG, destacando cómo los datos y el control fluyen a través de tu programa.

¿Cómo empezar?

Para comenzar el análisis, selecciona los archivos que desees analizar utilizando los botones de abajo y presiona Analizar.

(QCPG soporta exclusivamente archivos .qasm y .py escritos utilizando la biblioteca Qiskit.)
